# Imagens da landing page

Esta pasta é a `public/images` do Next.js. Os arquivos são referenciados como `/images/nome-do-arquivo`.

| Arquivo | Uso |
|---------|-----|
| `pilartec-logo.svg` ou `pilartec-logo.png` | Logo da Pilar Tec (header e hero) |
| `cronoar.png` (ou .svg) | Logo da Cronoar (parceiros) |
| `domdchef.png` (ou .svg) | Logo da Domdchef (parceiros) |
| `sgaagro.png` (ou .svg) | Logo do SGA Agro (parceiros) |

Se as suas imagens estão em outra pasta chamada `images`, copie-as para **pilartec-site/public/images/** (ou use os mesmos nomes acima).
